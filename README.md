# Галия Багдат — персональный сайт-визитка

Одностраничный имиджевый сайт эксперта по коучингу и обучению руководителей **Галии Багдат** (Нурбекова Галия Багдатовна). Реализован по [PRD.md](PRD.md).

## Особенности

- **Один HTML-файл** ([index.html](index.html)) со встроенными стилями и скриптами, без бэкенда.
- **Три языка** RU / KZ / EN с мгновенным переключением без перезагрузки.
- Премиум-дизайн: Cormorant Garamond + Manrope, изумрудно-золотая палитра, мягкие световые градиенты.
- Анимации появления при прокрутке с учётом `prefers-reduced-motion` и страховкой видимости через ~3 секунды.
- Сворачиваемый блок карьеры, демо-форма заявки, кликабельные прямые контакты.
- Адаптивная вёрстка (телефон / планшет / десктоп).

## Изображения

Положите эти файлы рядом с `index.html` — они подхватятся автоматически. Пока файлов нет, на их месте показываются аккуратные плейсхолдеры.

| Файл | Назначение |
|------|-----------|
| `photo-portrait2.jpg` | студийный портрет (Hero) |
| `portrait.jpg` | портрет в синем пиджаке (О себе) |
| `photo-pavlodar.jpg` | открытие филиала ICU в Павлодаре (галерея) |
| `photo-judo.jpg` | Qazaqstan Barysy Grand Slam (галерея) |

## Локальный просмотр

Откройте `index.html` в браузере — внешних зависимостей, кроме Google Fonts, нет.

## Публикация

Сайт развёрнут на GitHub Pages из ветки `main` (корень репозитория).
